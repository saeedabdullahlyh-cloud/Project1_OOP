# Pac-Man Game (Java OOP)

 Description
This project is a console-based Pac-Man game developed using Object-Oriented Programming (OOP) in Java. The player collects food while avoiding a ghost in a maze.

---

 Features
- Maze-based movement
- Player control (W/A/S/D)
- Random ghost movement
- Collision detection (Game Over)
- Score system

---

 OOP Concepts Used
- Encapsulation
- Aggregation
- Composition
- Abstraction

---

 Classes
- PacmanGame
- GameBoard
- Pacman
- Ghost
- Food

---

 How to Run
javac PacmanGame.java  
java PacmanGame
